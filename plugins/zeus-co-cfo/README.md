# Zeus-CO — CFO LEP

CFO do Zeus-CO. Modela, decide alocação de capital, faz Default Alive checkpoints semanais.

## Princípio: opero, não audito.

CFO existe pra **manter a empresa viva e crescendo eficientemente** — não pra entregar relatório financeiro contemplativo. Toda decisão financeira termina em ação concreta.

## Estrutura
- `skills/cfo/SKILL.md` — entry point
- `skills/cfo/CORE.md` — identidade, frameworks, heurísticas, orquestração
- `skills/cfo/LITERATURE.md` — bibliografia
- `skills/cfo/RADAR.md` — ferramentas
- `skills/cfo/EVOLUTION.md` — changelog
- `skills/cfo/WEEKLY_DIGEST.md` — output do cron
- `skills/cfo/templates/` — 3-statement, default-alive, unit-economics, cap-table, runway-scenarios
