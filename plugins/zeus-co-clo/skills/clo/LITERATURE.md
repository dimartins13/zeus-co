## Camada 4 — Brasil regulatório por setor (essencial)

### iGaming (KingPanda)
- **Lei 14.790/2023** — marco regulatório
- **Portarias SECAP/MF** — operacionalização
- **Licença**: R$30M outorga, válida 5 anos, renovável
- **Obrigações**: KYC obrigatório (Lei 9.613/98 lavagem), jogo responsável, % canalizado para esporte/saúde, idade mínima 18, vedação publicidade juvenil
- **Tributação específica** — IRPJ + CSLL + CIDE 18% sobre GGR

### Audiovisual / NFT (Crazyflips)
- **Lei 8.685/93** — incentivos audiovisuais (Art. 1º A: investimentos via FUNCINES)
- **ANCINE** — Certificado de Registro de Título (CRT) obrigatório
- **Copyright Lei 9.610/98** — direito autoral
- **NFT como security**: CVM análise caso a caso. Se token dá direito a participação econômica = security regulada.

### E-commerce / Moda (2ndStreet)
- **Decreto 7.962/13** — direitos consumidor e-commerce BR
- **CDC Lei 8.078/90** — direito troca/devolução em 7 dias
- **PROCON estadual**
- **Reverse logistics**: Lei 12.305/10 obrigatória pra marcas
- **Segunda mão moda**: Resolução CONFAZ pode aplicar ICMS-ST específico
- **Consignação celebrity**: contrato comercial específico, % comissão, exclusividade, uso de imagem (Lei 9.610/98 + direito personalidade)

### Saúde digital (TarjaPreta)
- **CFM Resolução 2.314/22** — telemedicina
- **ANVISA RDC 657/22** — Software como Dispositivo Médico (SaMD)
- **LGPD dados sensíveis** — saúde mental = categoria sensível, base legal restrita

### LGPD geral (todas empresas)
- **Lei 13.709/18 + decretos**
- **DPO obrigatório**: empresa que trata dados em escala ou sensíveis
- **Política de Privacidade + Termos de Uso** — obrigatórios em qualquer site/app
- **RIPD** — Relatório de Impacto à Proteção de Dados — quando alto risco
- **Multas**: até 2% faturamento ou R$50M por infração

### Fintech (cross-aplicável)
- **BACEN Circular 4.480** — PIX
- **Resolução BCB 80/21** — IF não bancárias
- **CVM** — se há valor mobiliário envolvido
