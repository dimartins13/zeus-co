# CORE — Regulatório Setorial
> **Mapeio. Notifico gate. Indico filing.**

## iGaming (KingPanda) — CRÍTICO
- **Lei 14.790/2023** — marco regulatório
- **SECAP/MF** — concedente da licença
- Outorga: R$30M (5 anos)
- Obrigações: KYC (Lei 9.613/98), AML, jogo responsável, % canalizado p/ esporte/saúde, idade mínima 18, vedação publicidade juvenil
- IRPJ + CSLL + CIDE 18% sobre GGR
- **Sem licença = operação criminal** — CEO + CLO + CFO orquestram cronograma

## Audiovisual (Crazyflips) — IP cross
- **Lei 8.685/93** — incentivos audiovisuais (FUNCINES)
- **ANCINE** — Certificado de Registro de Título (CRT) obrigatório
- **Lei 9.610/98** — direito autoral
- CONDECINE (incidência + isenção pra incentivos)
- Coprodução internacional regras específicas

## Saúde Digital (TarjaPreta)
- **CFM Resolução 2.314/22** — telemedicina (se houver acompanhamento)
- **ANVISA RDC 657/22** — Software como Dispositivo Médico (SaMD)
- **LGPD reforçada** — saúde mental = dado sensível
- Se não fizer claim médico (só motivacional) — ANVISA não aplica diretamente

## E-commerce (2ndStreet, Ventage, Creatina)
- **Decreto 7.962/13** — direitos consumidor e-commerce
- **CDC Lei 8.078/90** — troca/devolução em 7 dias
- **PROCON estadual** — 7 dias úteis pra resposta
- **Reverse logistics** — Lei 12.305/10 obrigatória
- **Marketplaces** (2ndStreet) — regras específicas de transparência consignação

## Suplementação (Creatina se confirmar)
- **ANVISA RDC 243/2018** — suplementos alimentares
- Categorias: vitaminas, minerais, aminoácidos, etc
- Notificação prévia (não registro) — mais ágil que medicamento
- Rotulagem obrigatória + alegações restritas (não terapêuticas)
- Boa Práticas Fabricação (BPF) certificada

## Fintech (futuro)
- **BACEN Circular 4.480** — PIX
- **Resolução BCB 80/21** — IF não bancárias
- **CVM** — se valor mobiliário envolvido
- **Lei 13.506/17** — sanções

## Heurísticas
- Regulação É gate de operação — não opcional
- Filing inicial sempre com escritório especializado
- Acompanhamento mensal das atualizações ANPD/ANVISA/SECAP/etc
- "Caixa preta" jurídico (advogado humano + Zeus-CO juntos)
