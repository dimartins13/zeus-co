# CORE — Trabalhista
> **CLT pro core. PJ pra C-level. Sempre estruturado.**

## Modalidades BR
- **CLT:** subordinação + habitualidade + onerosidade + pessoalidade. Encargos ~80% sobre salário.
- **PJ:** autônomo (sem 4 elementos acima). MEI, ME, EPP. Mais barato mas exige estrutura.
- **MEI:** simples + barato. Limite R$81k/ano.
- **Estagiário:** Lei 11.788. Termo de Compromisso obrigatório, supervisor formado, 6h/dia max.
- **Aprendiz:** Lei 10.097. 14-24 anos, escola técnica. Cota 5-15% empresas médias+.

## Vesting / ESOP no Brasil
- Plano de Stock Option (não regulado especificamente — usar contrato civil)
- Strike price = preço do exercício (pode ser igual valuation atual)
- Vesting típico: 4y / 1y cliff
- Tributação: ganho de capital se opção, ou rendimento se SAR
- ATENÇÃO: ESOP pra CLT pode virar verba trabalhista (jurisprudência inconsistente)

## Heurísticas
- C-level: PJ via PF/holding (limita exposição CLT)
- Devs senior: PJ aceitável se autonomia real
- Equipe core: CLT (segurança jurídica)
- Demissão sem justa: aviso prévio + multa FGTS 40% + verbas rescisórias
- Demissão por justa: documentação ROBUSTA (sem ela = reverte)

## Demissão checklist
1. Motivo documentado
2. Advertências prévias (se aplicável)
3. Cálculo verbas rescisórias (tem online)
4. Carta demissão + recibo
5. Exame demissional
6. Homologação sindical (se >1y CLT)
7. Atualizar 1Password / acessos / contas IMEDIATAMENTE
