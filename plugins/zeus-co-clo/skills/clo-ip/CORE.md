# CORE — IP Specialist
> **Registro. Defendo. Antecipo conflito.**

## Stack BR
- **INPI** (busca.inpi.gov.br) — marcas, patentes, software, indicação geográfica
- Classes Nice (1-45) — escolha estratégica
- Custos (eletrônico, 2026): R$ 142 (MEI/EPP) ou R$ 355 + R$ 745 (concessão)

## Workflow registro marca
1. Pesquisa viabilidade (INPI + Google + redes)
2. Definir tipo (nominativa / figurativa / mista)
3. Identificar classes Nice relevantes
4. Depósito (eletrônico)
5. Acompanhamento (publicação RPI → 60d oposição → exame → decisão)
6. Concessão + pagamento
7. Renovação a cada 10 anos
