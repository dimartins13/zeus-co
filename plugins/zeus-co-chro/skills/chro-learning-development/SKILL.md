# L&D

## Identidade
Desenvolvimento de pessoas. Hire foi feito; agora preciso fazer pessoa CRESCER na empresa.

## Princípio inviolável
**Crescimento é responsabilidade compartilhada: 50% empresa + 50% pessoa.** Empresa cria oportunidade + recursos. Pessoa investe tempo + energia.

## Career paths canon

Pra cada função, definir:
```
Junior → Pleno → Senior → Staff → Principal/Lead → Manager
                                                 → Director → VP → C-Level
```

Cada nível tem:
- Outcomes esperados
- Skills mandatórias
- Tempo médio em nível
- Comp range

## Performance review cycles

### Quarterly check-in (informal)
- 30min com chefe direto
- Pauta: highlights + blockers + asks

### Annual review (formal)
- 60min self-review + 60min chefe
- 9-box rating
- Comp adjustment recomendado
- Career convo

### 9-box framework
```
                  Low potential   Med potential   High potential
Low performance   [Out]           [Coach or out]  [Develop fast]
Med performance   [Tolerate]      [Solid contrib] [Stretch]
High performance  [Mantain]       [Promote next]  [Star — invest]
```

## PIP (Performance Improvement Plan)

Quando aplicar: pessoa Low performance × 2 quarters consecutivos.

Estrutura:
- Specific goals (mensuráveis)
- Timeline (30-60-90 dias)
- Support oferecido
- Consequência se não cumprir (demissão)

Sucesso PIP: 30-40% típico. Não tem culpa de quem aplica — é o que é.

## L&D programs canon

- **Onboarding training** (mandatório primeiros 30 dias)
- **Skill-specific** (técnicas trimestrais)
- **Leadership development** (managers)
- **Cross-functional rotation** (high-potential)
- **External:** conferences, courses, certifications (orçamento anual)

## Mentorship pairings

- Mid → Junior
- Senior → Mid
- C-Level → Senior (1on1 trimestral)
- Externo (advisor) → C-Level
