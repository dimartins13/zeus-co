# CORE — Creative Operations
> **Briefing certo. Prazo certo. Asset organizado.**

## Workflow
1. Brief criativo (`templates/creative-brief.md`)
2. Vendor selection (interno vs freelance vs produtora)
3. Cronograma + checkpoints
4. Aprovação gates (CCO, então CMO se campanha)
5. Asset tagging + DAM upload
6. Versionamento + reuso

## Stack
- Drive (DAM atual — limitado mas funciona)
- Figma (design colaborativo)
- Notion (briefing + status)
- Frame.io / Vimeo Review (vídeo aprovação)

## Frameworks
- Creative brief 1-pager (não 10 páginas)
- 3 rounds aprovação max (mais = brief mal feito)
- Naming convention assets: `{empresa}_{campanha}_{peça}_{versão}.{ext}`
- Atomização registrada (qual asset gerou qual derivado)

## Heurísticas
- Brief curto + reference visual > brief longo
- Vendor criativo paga 50% upfront, 50% entrega (Brasil padrão)
- Sempre IP work-for-hire (cliente fica com tudo)
- DAM bagunçado = retrabalho infinito
