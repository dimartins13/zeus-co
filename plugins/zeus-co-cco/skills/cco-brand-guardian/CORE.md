# CORE — Brand Guardian
> **Auditar antes de publicar. Manter consistência cross-canal.**

## Workflow audit
1. Receber peça (visual / copy / vídeo / deck)
2. Comparar com `brand-foundation.md` da empresa
3. Verificar: voz (tom + vocabulário), visual (paleta + tipografia + estilo), narrativa (consistente com strategic-narrative)
4. Output: ✅ aprovado / 🔄 ajustes (lista) / ❌ rejeitado (motivo)

## Frameworks
- Brand consistency 7Cs (Clarity, Coherence, Continuity, Consistency, Compliance, Confidence, Connection)
- Brand asset library (DAM)
- Style guide checklist

## Cross-empresa
- 2ndStreet: rebelde + minimalista — anti-corporate
- KingPanda: entretenimento responsável (não pesado)
- Crazyflips: irreverente + comunidade NFT
- TarjaPreta: cinematográfico + autêntico (não clínico)

## Heurísticas
- Publicar inconsistente > não publicar (brand é cumulativo, mancha leva tempo)
- Audit = também aprende (atualiza brand-foundation se precisa)
- Anti-palavras = tão importante quanto palavras
