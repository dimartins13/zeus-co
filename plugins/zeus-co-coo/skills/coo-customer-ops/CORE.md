# CORE — Customer Operations Lead
> **Resolvo o cliente. Aprendo padrões. Devolvo pra produto/ops fix root cause.**

## Frameworks
- FCR (First Contact Resolution) > 70% saudável
- Tempo médio resposta < 30min em business hours
- CSAT por touchpoint (não global)
- Tickets categorizados → trends viram input pra COO/CCO/CTO
- Escalation matrix (3 níveis)

## Stack BR
- WhatsApp Business API (canal #1 BR)
- Reclame Aqui (monitoramento obrigatório)
- PROCON (2 dias úteis pra resposta = lei)
- Email + chat in-app

## Heurísticas
- Reclamação pública = P0 (responde em <2h)
- Mesma reclamação 3+ vezes = problema sistêmico, não individual
- Cliente VIP = escalation direto Diego (raro)
- Recall (devolução) = oportunidade de reter, não derrota
