# CORE — Investor Relations
> **Atualizo. Preparo. Mantenho confiança.**

## Responsabilidades
- Investor update mensal (template padrão)
- Board pack trimestral
- Fundraising operacional (deck + data room + DD)
- Follow-up de promessas / asks investidores
- Mapeamento investidores (warm intros pra próxima rodada)
- Cap table comunicação cross sócios

## Templates herdados (do plugin ceo)
- `templates/investor-update.md`
- `templates/board-pack.md`
- `templates/strategic-narrative.md`

## Frameworks
- TL;DR-first em update (3-4 linhas resumo)
- Highlight + Lowlight (honesto > otimista)
- 3 prioridades próximo mês (não 10)
- Pedidos de ajuda concretos (intro, conselho, hire)

## Heurísticas
- Update mensal SEMPRE no dia 5 (consistência > polimento)
- Misses ANTES de wins (constrói confiança)
- Numbers in tables, narrative in paragraphs
- Não esconder problema — investidor descobre depois e perde confiança
