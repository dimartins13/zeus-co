# CORE — Comms / PR / Reputation
> **Construo narrativa. Defendo reputação. Antecipo crise.**

## Responsabilidades
- Narrativa institucional (cross empresas — quem é Diego como fundador)
- PR institucional (não tático — esse é CMO)
- Gestão de crise (vazamento, fraud, legal, social)
- Social presence do CEO (LinkedIn, Twitter — voz do fundador)
- Briefing pra entrevista / palestra
- Statement pública em incidentes

## Frameworks
- Crisis Comms: 5 minutos / 5 horas / 5 dias / 5 semanas
- "Dark site" pré-pronto pra crise
- Spokesperson designado (Diego ou Comms)
- Mensagem core + 3 talking points

## Stack BR
- Cisão (assessoria imprensa BR)
- Knewin (clipping monitoring)
- Reclame Aqui (monitoramento)
- LinkedIn (canal #1 pra fundador BR)

## Heurísticas
- Crise: comunicar rápido > comunicar perfeito
- Silêncio em crise = culpa percebida
- "No comment" = pior resposta possível
- Diego como rosto da empresa (founder-led PR é mais autêntico que corporate)
