# PIPELINE — Entrepreneurship / Venture Studio

> Diego é fundador serial. Empresa Diego = portfolio de empresas, não 1 empresa.
> Esse departamento opera a CRIAÇÃO de novas empresas + gestão de portfolio.

---

## Pipeline (8 fases)

### Fase 0 — Descoberta Interna (raiz Projects)
- Estado atual portfolio (5 empresas + infra)
- Capacidade tempo Diego

### Fase 1 — Idea pipeline
Owner: `entrepreneurship` + cross `cino`
- Pipeline de ideias (10-50)
- Filtros: mercado + Diego fit + uniqueness
- Output: `_Areas/Entrepreneurship/idea-pipeline.md`

### Fase 2 — Validation sprint
Owner: `new-co-design`
- 2-4 semanas validar Top 3 ideas
- Lean Canvas + 20 entrevistas + smoke test
- GO/NO-GO decision
- Output: `_Areas/Entrepreneurship/validations/<idea>/`

### Fase 3 — New Co design
Owner: `new-co-design`
- Modelo de negócio detalhado
- Founding team (Diego solo? Co-founder?)
- Cap table inicial
- Estrutura societária (LTDA / SA / Holding)
- Initial funding (bootstrap / family/friends / pre-seed)
- Setup técnico (CNPJ + bank + plataforma)
- Output: `_Areas/Entrepreneurship/new-cos/<empresa>/birth-memo.md`

### Fase 4 — Launch
Owner: `new-co-design` → handoff `zeus-co-ceo` da empresa nova
- Day 1 checklist
- Estrutura Zeus-CO replicada na empresa nova
- C-Levels (LEPs) ativados
- Output: empresa em `~/Documents/Claude/Projects/<empresa>/`

### Fase 5 — Portfolio governance
Owner: `portfolio-allocation`
- Alocação tempo Diego cross-portfolio (semanal/mensal)
- Three Horizons portfolio-level
- Performance review portfolio mensal
- Output: `_Areas/Entrepreneurship/portfolio-allocation.md`

### Fase 6 — Co-founder + advisor recruitment
Owner: cross `entrepreneurship` + `chro-recruiting` + `board-advisors-management`
- Quando trazer co-founder em new co
- Equity split decisions
- Vesting + cliff alinhados

### Fase 7 — Exit strategy
Owner: `exit-strategy`
- Pra cada empresa: M&A / IPO / lifestyle / kill
- Cada estágio tem exit window típica
- Decisão Type 1 quando aplicável

### Fase 8 — Kill or scale (decisão semanal por empresa)
Owner: `portfolio-allocation`
- 3 estados: ativo / sleep / kill
- Critério: Diego tempo investido vs traction
- Output: log mensal `portfolio-decisions-YYYY-MM.md`

---

## Princípio inviolável

**Foco > diversificação.** Maioria dos fundadores seriais que falham têm 5+ empresas estagnadas. 1-2 com tração explosiva > 5 estagnadas.
