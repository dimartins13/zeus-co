# zeus-co-universidade

A **Universidade Zeus-CO** empacotada como plugin: a biblioteca de conhecimento (fonte + princípio + case) que os especialistas do Zeus-CO consultam antes de opinar.

## O que tem
- Skill `/universidade` — consulta a biblioteca e responde **citando a ficha-fonte**, distinguindo consenso de disputa e dizendo "não está na biblioteca" quando não sabe.
- `skills/universidade/references/biblioteca/` — 3 faculdades auditadas:
  - **CMO** — marketing, growth, tráfego pago, publicidade (técnicas + swipe files + cases), branding.
  - **CFO** — finanças de fundador: unit economics, runway/Default Alive, viabilidade, cap table, tributário BR.
  - **CEO** — estratégia, PMF, liderança, sócios & governança.
