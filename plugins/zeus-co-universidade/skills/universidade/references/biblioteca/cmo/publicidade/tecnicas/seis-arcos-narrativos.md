# Toda peça premiada tem arco; 6 se repetem

**TL;DR:** Narrativa é arco — sem arco, é decoração. Os premiados reusam 6 estruturas de arco. Escolha a que o insight pede.

1. **Problema → Inversão → Resolução** — NZ Herpes: você tem (problema) → "melhor lugar pra ter" (inversão) → comunidade vive bem (resolução).
2. **Defeito → Demonstração → Aceitação** — Heinz Smiles: ketchup mancha → vira sorriso → afeto.
3. **Velho → Atualizado → Renovado** — L'Oréal *Ilon Specht*: slogan antigo → história por trás → frase renasce.
4. **Lateral → Centro → Expansão** — Microsoft *ADLaM*: língua marginalizada → digitalizada → integrada.
5. **Cotidiano → Heroico → Cotidiano** — IKEA *Wonderful Everyday*: manhã banal → arquitetada → café que importa.
6. **Hijack → Apropriação → Cultura** — DoorDash *All the Ads*: Super Bowl → entrega tudo → categoria nova.

**Como usar:** depois de fixar insight e Big Idea, pergunte "qual arco carrega esta ideia num filme de 60s E num OOH?". O arco tem de sobreviver à troca de formato.

**Limite:** arco é esqueleto, não roteiro — dois cases do mesmo arco podem parecer nada iguais. E arco sem insight verdadeiro (Camada 1) é forma vazia.
