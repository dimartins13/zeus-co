# Crescimento sustentável vem de loops, não de funis

**TL;DR:** O funil trata aquisição como um evento linear que termina no cliente; o loop trata cada cliente convertido como insumo que gera o próximo — é isso que compõe crescimento ao longo do tempo.

O modelo de funil (topo → meio → fundo) descreve bem uma campanha, mas engana como modelo de crescimento porque é linear e tem fim: cada novo cliente exige um novo gasto de aquisição, sem reinvestir a energia gerada. O loop reformula o sistema como um ciclo fechado em que a saída realimenta a entrada. Exemplos de mecanismo: um usuário cria conteúdo que atrai novos usuários (loop de conteúdo/UGC), convida contatos (loop viral), ou paga uma assinatura cuja margem financia mais aquisição paga (loop pago). O crescimento composto aparece quando cada volta do loop deixa o próximo giro mais barato ou mais forte.

Para o fundador, a virada prática é parar de perguntar "qual meu funil de conversão?" e passar a perguntar "qual é o loop que me faz crescer sem depender de gastar cada vez mais?". Loops são mais difíceis de construir que campanhas, mas são o que dá defensibilidade e economia de escala.

Condição de contorno: nem todo negócio tem loop forte, e loop não substitui aquisição paga — muitas vezes eles coexistem, com o pago financiando o início até o loop orgânico ganhar tração. Loops também degradam (saturação de rede, fadiga de convite, queda de alcance orgânico), então precisam de manutenção, não são perpétuos. A ferramenta certa depende do modelo: negócios sem efeito de rede nem conteúdo podem crescer bem por loops puramente econômicos (LTV reinveste em aquisição). O erro a evitar é desenhar todo o crescimento como funil e nunca perguntar o que realimenta o sistema.

## Fontes
- Brian Balfour / Reforge, *Growth Loops* (ensaios) — formaliza a substituição do funil pelo loop como unidade de crescimento.
- Andrew Chen, *The Cold Start Problem* — loops de rede e sua saturação ao longo do tempo.
