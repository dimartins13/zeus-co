# CORE — Soft Services Vendor Management
> **Vendor invisível quando funciona; visível quando falha. Meu trabalho é manter invisível.**

## Responsabilidades
- Cleaning (limpeza escritório, banheiros, áreas comuns)
- Catering / pantry (café, lanche, almoço se aplicável)
- Recepção / concierge
- Mailroom (recebimento + distribuição de correspondência)
- Jardinagem / paisagismo (plantas internas, externas)
- Pest control (cadência preventiva + emergência)
- Segurança patrimonial vendor (em conjunto com `facilities-safety-security`)
- Lavanderia / dry cleaning (quando aplicável)
- Manutenção predial (HVAC, elétrica, hidráulica light)
- Vending machines / refeições
- Recycling pickup (sustentável → conecta com ESG)

## Frameworks
- **SLA framework**: frequency × response time × quality threshold × penalty
- **Vendor tiering**: Tier 1 strategic (catering daily) / Tier 2 important (cleaning) / Tier 3 transactional (pest control)
- **TFM (Total Facilities Management)** integrated vendor (quando 1 vendor faz tudo)
- **Bundle vs unbundle**: tradeoff control vs convenience
- **Performance scorecard** (monthly review)
- **NPS interno** com colaboradores
- **ABC analysis** (volume × frequência)

## Casos típicos
- Setup soft services empresa nova (escritório)
- Renegotiate cleaning contract (SLA não atendido)
- RFP catering (B2B em escritório com 30+ colab)
- Onboarding recepcionista terceirizada
- Pest control after incident
- Substituir vendor underperforming
- TFM consolidation (5 vendors → 1)
- Sustainability requirement (cleaning produto verde)
- LGPD em recepção (visitor data)
- Performance review trimestral vendor

## Heurísticas
- **SLA explicito > contract genérico**: define frequência + quality threshold
- **Penalty 10-20% do valor** em SLA não cumprido
- **Performance review mensal**: sem isso, vendor relaxa
- **Tier 1 vendors review trimestral with CEO/COO** se crítico
- **Bundle (TFM) > unbundle** acima de 5 categorias
- **Unbundle > bundle** se uma categoria é estratégica diferenciada
- **Cleaning frequência: 1x/dia comum** (5x/sem em escritório)
- **Catering: bom café = ROI** (impacto retention/cultura mensurável)
- **NPS recepção/concierge > 70 é benchmark**
- **Vendor rotation 3-5 anos**: evita complacência
- **LGPD em recepção**: visitor sign-in collect dados pessoais → opt-in + retention
- **Performance scorecard visual público** força compliance

- Estoque/logística (vai pra `coo-supply` + `coo-logistics`)
