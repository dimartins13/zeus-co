# CORPUS — Cases BLEND (Misturar / Combinação)

> 60 cases canônicos cross-indústria onde 2+ conceitos foram fundidos em terceiro novo com propriedades emergentes. Skill consulta durante geração de Blend.

> Formato: **Nome** (setor) — `Input 1 + Input 2`. **Generic Space**. **Propriedade emergente**.

## Como skill usa

1. **Em geração:** ao definir Input 2 (domínio distante), puxa 3-5 cases pra ver "como ficou quando alguém combinou X + Y" 
2. **Validação:** se ideia gerada tem case análogo no corpus, ganha tração defensiva
3. **Anti-padrão:** cases óbvios demais (mesma combinação) → skill propõe variação
4. **Learning:** cases Zeus-CO validados (Diego aprovou e implementou) viram parte do corpus interno
