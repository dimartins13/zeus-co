# CORPUS — Cases BREAK (Quebrar / Desconstrução)

> 60 cases canônicos cross-indústria onde uma assunção fundadora foi quebrada (eliminada, modificada radicalmente ou invertida). Skill consulta durante geração de Break.

> Formato: **Nome** (setor) — `Constraint quebrado`. **Mecanismo**. **Espaço novo**.

---

## 1. Modelo de negócio (clássicos)

### Netflix (1997) — `Quebrou: loja física + multa atraso`
- **Mecanismo:** aluguel postal → depois streaming. Eliminou loja, eliminou multa, eliminou "data de devolução".
- **Espaço novo:** assinatura mensal flat. Conteúdo sob demanda, ilimitado.

### Spotify (2008) — `Quebrou: comprar música como propriedade`
- **Mecanismo:** assinatura mensal → acesso ilimitado. Quebrou compra individual, MP3 download, álbum como unidade.
- **Espaço novo:** "música como serviço de utilidade", playlists > albums.

### IKEA — `Quebrou: móvel pronto + montagem por loja`
- **Mecanismo:** móvel flat-pack + montagem pelo cliente. Quebrou logística + serviço pós-venda.
- **Espaço novo:** preço acessível + showroom-as-storage + DIY mandatório.

### Cirque du Soleil — `Quebrou: animais + múltiplos picadeiros + comida`
- **Mecanismo:** removeu animais, removeu shows simultâneos, removeu venda de pipoca dentro. Reposicionou pra adulto upscale.
- **Espaço novo:** "circo como teatro" — preço de Broadway, narrativa, sem stigma.

### Costco — `Quebrou: margem alta no varejo`
- **Mecanismo:** margem ~10% (vs varejo 30-40%) + receita via membership annual fee. Quebrou modelo financeiro varejo.
- **Espaço novo:** atacado pra consumidor com fidelização paga.

### Trader Joe's — `Quebrou: variedade massiva no supermercado`
- **Mecanismo:** ~4000 SKUs (vs ~50000 supermercado típico). Quebrou "mais é melhor".
- **Espaço novo:** curadoria como diferencial, escala pequena propositada.

### Aldi/Lidl — `Quebrou: marca + ambiente loja + serviço`
- **Mecanismo:** marcas próprias quase só, layout austero, sem ensacador. Quebrou padrões varejo.
- **Espaço novo:** preço imbatível via remoção de tudo "experiencial".

### Southwest Airlines — `Quebrou: hub-and-spoke + classes`
- **Mecanismo:** rotas ponto-a-ponto, sem classes, sem reserva de assento, 1 modelo de avião. Quebrou complexidade.
- **Espaço novo:** low-cost airline original — modelo replicado globalmente.

### Ryanair — `Quebrou: tudo incluso no preço`
- **Mecanismo:** preço base mínimo + tudo a la carte (bagagem, assento, check-in). Quebrou bundling.
- **Espaço novo:** ultra low-cost com receita em ancillary.

### Tesla direct-to-consumer — `Quebrou: revendedor concessionária`
- **Mecanismo:** vende direto, lojas-conceito sem barganhar preço. Quebrou modelo distribuição.
- **Espaço novo:** preço transparente + experiência produto > experiência loja.

## 2. Tech / Software

### Linux/Open Source — `Quebrou: software como propriedade fechada`
- **Mecanismo:** código aberto + licença reciprocidade (GPL). Quebrou propriedade intelectual privada.
- **Espaço novo:** colaboração global como modo de produção.

### Wikipedia — `Quebrou: enciclopédia paga + autoria credentialed`
- **Mecanismo:** crowd-edit + gratuito + verificação social. Quebrou autoridade editorial.
- **Espaço novo:** conhecimento como bem comum auto-gerido.

### Twitter (2006) — `Quebrou: post longo blog`
- **Mecanismo:** limite 140 char. Quebrou expectativa de profundidade.
- **Espaço novo:** comunicação em pulsos curtos, real-time stream.

### Snapchat — `Quebrou: posts permanentes`
- **Mecanismo:** mensagens expiram. Quebrou arquivo digital permanente.
- **Espaço novo:** mídia efêmera, intimidade > pose curada.

### TikTok — `Quebrou: follow-based feed`
- **Mecanismo:** algoritmo dominado pela For You Page (não follows). Quebrou modelo Twitter/Insta.
- **Espaço novo:** descoberta > rede social — qualquer um pode viralizar.

### Slack — `Quebrou: email como comunicação corporativa`
- **Mecanismo:** chat por canal + thread + integração. Quebrou silos email.
- **Espaço novo:** comunicação assíncrona pública por contexto.

### Notion — `Quebrou: app por função (doc / planilha / database)`
- **Mecanismo:** blocos modulares que viram qualquer coisa. Quebrou software-por-tarefa.
- **Espaço novo:** software composicional generalist.

### Stripe — `Quebrou: integração de pagamento como projeto de semanas`
- **Mecanismo:** API design-first. Quebrou complexidade enterprise.
- **Espaço novo:** infraestrutura financeira "developer-friendly".

### Figma — `Quebrou: design tool desktop + arquivo individual`
- **Mecanismo:** browser + colaboração em tempo real. Quebrou padrão Adobe.
- **Espaço novo:** design como prática colaborativa contínua.

### Y Combinator — `Quebrou: VC tradicional (cheque grande, board seat)`
- **Mecanismo:** cheque pequeno + programa intensivo + comunidade. Quebrou estrutura.
- **Espaço novo:** acelerador como produto.

## 13. Negativo / Cuidado (Break mal-feito)

### Quibi — `Quebrou: vídeo livre vs paywall premium`
- **Mecanismo:** vídeos curtos mobile-only premium $4.99/mo. Quebrou modelo errado.
- **Espaço novo:** ❌ não havia demanda. Faliu em 6 meses.
- *Insight:* nem todo Break gera valor — pode quebrar coisa que cliente queria.

### New Coke (1985) — `Quebrou: fórmula histórica`
- **Mecanismo:** mudou fórmula. Quebrou identidade.
- **Espaço novo:** ❌ revolta do consumidor → voltou em 79 dias.
- *Insight:* Break pode violar contrato implícito com consumidor.

## Como skill usa

1. **Em geração:** lista assunções do briefing → busca cases com assunções análogas quebradas → propõe Break análogo
2. **Como precedente defensivo:** ideia gerada tem suporte de case real bem-sucedido
3. **Como stress-test:** se case análogo falhou (Quibi, New Coke), pesa contra Break proposto
