# Léxico cross-domain — ESPORTE

> Vocabulário e mecânicas esportivas como matéria-prima pra Bend/Blend.

## Dimensões centrais

- **Regra** (codificada, arbitrada)
- **Placar** (mensurável, comparável)
- **Tempo** (round, set, quarter, minuto)
- **Fase** (treino, classificatória, eliminatória, final)
- **Time** (solo, dupla, time, federação)
- **Intensidade** (resistência, força, explosão)
- **Performance** (PB, recorde, posição)
- **Adversário** (humano direto, contra tempo, contra natureza)

## Conceitos importáveis

| Conceito esportivo | O que carrega | Pra que serve fora |
|---|---|---|
| **Personal Best (PB)** | Recorde próprio independente de outros | Métrica de crescimento individual (gamification) |
| **Streak** | Sequência sem quebrar | Engagement loop, Duolingo, fidelidade |
| **Campeonato** | Série com classificação acumulada | Programa de fidelidade competitivo |
| **Underdog** | Quem ninguém esperava | Narrativa de marca não-líder |
| **Comeback** | Virada do perdedor | História de recuperação |
| **Aquecimento** | Preparação obrigatória pré-jogo | Onboarding, pré-venda |
| **Tempo extra / prorrogação** | Continua se empatado | Refund window, garantia estendida |
| **Pênaltis** | Decisão sob pressão isolada | Negociação 1-a-1 high-stakes |
| **MVP** | O melhor do jogo/temporada | Influencer principal, top customer |
| **Treinador / técnico** | Estrategista bastidor | Mentor, consultoria de operação |
| **Olheiro / scout** | Detecta talento bruto | Recrutamento, M&A discovery |
| **Federação** | Órgão regulador esporte | Regulação de categoria nova |
| **Fair play** | Conduta ética intrínseca | Cultura corporativa |
| **Time amador** | Joga por amor, não dinheiro | Comunidade UGC engajada |
| **Patrocínio** | Marca paga visibilidade em time/atleta | Sponsorship/endorsement |
| **Drama esportivo** | Narrativa do jogo em si | Storytelling de produto |
| **Camisa numerada** | Identidade visual + posição | Onboarding com role |
| **Vestiário** | Espaço privado time | Comunidade VIP fechada |
| **Hat-trick** | 3 ações marcadas seguidas | Achievement unlock |
| **Doping** | Trapaça invisível | Crise ética, fraude operacional |
| **Off-side / falta** | Quebra regra dentro do jogo | Trade-off entre vantagem e risco |

## Estruturas importáveis

| Estrutura | Lógica |
|---|---|
| **Liga** | Times mesma divisão jogam todos contra todos | Marketplace fechado |
| **Mata-mata** | Eliminatória sucessiva | Funil de seleção |
| **Round-robin** | Cada um joga com cada um | Avaliação 360 |
| **Suíço** | Pareamento por performance | Matchmaking algorítmico |
| **Pyramid challenge** | Quem ganha sobe | Hierarquia gamificada |

## Filosofias importáveis

- **"Just Do It" (Nike)** — agir sobre pensar
- **"No pain, no gain"** — dificuldade como combustível
- **"Train like you fight"** — treino realista
- **"Marginal gains" (Sky Cycling)** — melhorias 1% multiplicadas
- **"Process > outcome"** — foco no método não no resultado
- **"Iron sharpens iron"** — rivalidade fortalece
- **Sportsmanship** — vencer com classe
- **Hometown advantage** — contexto importa
- **The zone / flow** (Csikszentmihalyi) — estado de fluxo

## Como skill usa

- Quando briefing precisa **estrutura competitiva** ou **mensurável**
- Quando precisa **mecânica de engagement repetida** (streak, PB, championship)
- Quando precisa **narrativa underdog / comeback**
- Para **fitness/saúde** produtos com referências cruzadas
- Para **naming**: Apex, Streak, Sprint, Pivot, Captain, Coach, Anchor, Rally
