# Léxico cross-domain — GASTRONOMIA

> Vocabulário e mecânicas culinárias como matéria-prima pra Bend/Blend (especialmente em experiência sensorial, ritual, identidade cultural).

## Dimensões centrais

- **Sabor** (doce, salgado, amargo, ácido, umami)
- **Textura** (crocante, cremoso, gelatinoso, ar)
- **Temperatura** (gelado, ambiente, morno, quente, fumegante)
- **Tempo** (rápido, slow food, fermentado, envelhecido)
- **Origem** (regional, internacional, importado, terroir)
- **Técnica** (cru, cozido, grelhado, fermentado, defumado)
- **Ritual** (silêncio, comunal, religioso, lazy)
- **Estação** (sazonal, perene)
- **Hierarquia** (entrada-prato-sobremesa, harmonização)

## Conceitos importáveis

| Conceito gastronômico | O que carrega | Pra que serve fora |
|---|---|---|
| **Mise en place** | Tudo preparado antes de cozinhar | Pre-launch checklist, ops prep |
| **Mãe (mother sauce)** | Base de qual derivam outras | Plataforma + variações |
| **Terroir** | Sabor do lugar | Marca enraizada em geografia |
| **Maridagem** | Combinação calculada vinho + comida | Co-marketing intencional |
| **Tasting menu** | Sequência curada chef | Onboarding curado em camadas |
| **Carte (a la)** | Cliente monta | Self-service modular |
| **Slow food** | Anti-fast | Anti-conveniência intencional |
| **Fermentação** | Tempo + microorganismo = transformação | Crescimento orgânico passivo |
| **Defumação** | Sabor por exposição a fumaça lenta | Brand building lento via exposição cultural |
| **Reduction** | Concentrar via evaporação | Mensagem reduzida ao essencial |
| **Plating** | Apresentação visual no prato | UI / package design |
| **Amuse-bouche** | Aperitivo cortesia surpresa | Free sample, easter egg |
| **Petit four** | Doce final junto com café | Brinde de fechamento, after-purchase delight |
| **Sobremesa** | Doce no fim que muda memória | Encore positivo |
| **Pão na mesa** | Convite implícito | Onboarding gratuito |
| **Doggy bag** | Leva pra casa | "Save for later" |
| **Garçom invisível** | Serviço sem ser visto | UX seamless |
| **Receita de família** | Tradição passada | Heritage brand |
| **Comfort food** | Comida de conforto emocional | Produto refúgio |
| **Street food** | Acessível na rua | Marca democrática |

## Movimentos / Escolas

| Movimento | Origem | Importação cultural |
|---|---|---|
| **Nouvelle Cuisine (FR 1970s)** | Reação contra cozinha clássica pesada | Refresh contra excesso, leveza calibrada |
| **Molecular Gastronomy (Adrià)** | Ciência + cozinha | Tech aplicada em domínio tradicional |
| **Farm-to-table** | Ingrediente local rastreável | Supply chain transparente |
| **Slow Food (Petrini, IT 1986)** | Anti-McDonald's | Resistência calibrada a aceleração |
| **Nordic / New Nordic (Redzepi)** | Terroir nórdico extremo | Hyper-localismo como cred |
| **Cozinha de raiz BR (Atala)** | Ingredientes brasileiros profundos | Identidade BR comestível |
| **Sushi tradicional (Edomae)** | Ritual disciplinado mestre-aprendiz | Excelência via repetição quieta |
| **Vegana / Plant-based** | Sem produto animal | Marca de propósito sem perder produto |

## Tipologias importáveis

| Tipologia | Lógica | Aplicação |
|---|---|---|
| **Cantina** | Acessível, popular, regional | Produto mass market autêntico |
| **Brasserie** | Casual francês democrático | Premium acessível |
| **Bistrô** | Pequeno, íntimo, menu curto | Boutique brand |
| **Trattoria** | Familiar italiana | Marca-família |
| **Fine dining (3 estrelas)** | Ritual completo | Luxo experiencial |
| **Padaria de bairro** | Onipresente, café da manhã | Marca cotidiana invisível |
| **Boteco BR** | Cerveja gelada, salgadinho, conversa | Marca social comunitária BR |
| **Mercado de rua** | Caos curado | Marketplace caótico ordenado |
| **Açougue tradicional** | Produto raw, master visível | Heritage premium |
| **Cafeteria especialidade** | Single origin, terroir café | Marca artesanal nicho |
| **Food truck** | Móvel, sem chão, ágil | Pop-up, brand temporária |
| **Buffet por kg** | Cliente serve, paga por peso | Self-service, paga-pelo-que-usa (SaaS!) |
| **Rodízio churrasco** | Tudo que pode comer, preço fixo | Subscription unlimited |
| **Restaurante 1 mesa** | Reserva mensal, exclusivo | Edição super limitada |

## Como skill usa

- Quando briefing precisa **sensorialidade** ou **identidade cultural** comestível-like
- Quando precisa **estrutura de ritual** (tasting menu, mise en place, brinde)
- Quando precisa **calibrar acessível ↔ premium** (cantina → 3 estrelas)
- Para **marca comportamental** com vocabulário sensorial
- Para **naming**: Umami, Mise, Terroir, Reductio, Sazon, Fermento, Petit
