# Léxico cross-domain — MÚSICA

> Vocabulário, estruturas e mecânicas do domínio musical disponíveis pra Bend/Blend em outros domínios. Cada conceito traz **dimensão** (Gärdenfors) + **importação típica** (pra que serve em outro setor).

## Dimensões centrais

- **Ritmo** (cadência, tempo, pulse, BPM, syncopation)
- **Melodia** (linha tonal, hook, refrão)
- **Harmonia** (acordes, tensão/resolução, dissonância)
- **Timbre** (cor sonora, instrumento, voz)
- **Dinâmica** (volume, intensidade, contraste forte/fraco)
- **Andamento** (velocidade emocional)
- **Registro** (grave/agudo)
- **Estrutura** (intro/verso/refrão/ponte/coda)
- **Gênero** (classificação cultural)

## Conceitos importáveis

| Conceito musical | O que carrega | Pra que serve fora da música |
|---|---|---|
| **Hook** | Frase memorável repetida | Tagline, gancho de campanha, refrão de pitch |
| **Crescendo** | Construção gradual de intensidade | Lançamento em camadas, narrativa de pitch deck |
| **Contraponto** | 2 linhas independentes em harmonia | Posicionamento competitivo dual |
| **Sample** | Pedaço de música existente reusado | Brand collab, referência cultural em comunicação |
| **Remix** | Reinterpretação de original | Reposicionamento, pivot de produto preservando equity |
| **Cover** | Versão própria de música de outro | Adaptação local de modelo importado |
| **Pausa / silêncio** | Espaço entre notas | Ritmo de release schedule, white space em design |
| **Improvisação** | Estrutura aberta dentro de framework | Operação ágil dentro de SOP |
| **Riff** | Padrão melódico curto identitário | Logo sonoro, padrão visual identitário |
| **Build / Drop** | Tensão → liberação (EDM) | Pre-launch buzz → reveal |
| **Featuring** | Convidado em faixa de artista | Brand collab, co-marketing |
| **Acústico** | Versão despojada | "Behind the scenes", versão raw de produto |
| **Cassette / Vinyl** | Mídia física nostalgia | Embalagem premium, edição limitada física |
| **Mixtape** | Coletânea curada por DJ | Curated content drop, sample box |
| **Concerto ao vivo** | Performance única irrepetível | Evento experiencial, drop limitado |
| **Setlist** | Sequência de músicas escolhida | Roteiro de evento, ordem de menu |
| **Encore** | "Bis" pedido pela plateia | Engagement loop, "voltou por demanda" |
| **Genre crossover** | Mistura de gêneros (jazz + hip-hop) | Posicionamento híbrido categoria |
| **A capella** | Voz sem instrumento | Produto core sem features adicionais |
| **Beatbox** | Voz simula bateria | Substituir tech por humano performático |

## Padrões estruturais

### Estrutura ABA (verso-refrão-verso)
- **Importação:** narrativa de email marketing (problema-promessa-problema) ou pitch (contexto-virada-contexto)

### Build-up + Drop (EDM)
- **Importação:** Apple Keynote — "one more thing" como o drop

### Loop infinito
- **Importação:** Spotify Discover Weekly loop, Duolingo streak

### Bridge (ponte)
- **Importação:** transição em landing page, "antes que você desça, espera..."

## Tribos / Cenas (Gärdenfors centroides)

| Cena | Estética distinta importável |
|---|---|
| **Punk DIY** | Anti-corporativo, low-fi, anti-polish (Liquid Death) |
| **Hip-hop drill** | Street, autêntico, lyrics densas |
| **Indie folk** | Acústico, intimidade, vulnerabilidade |
| **Metal extremo** | Brutalidade estética, comunidade fechada, identidade tribal |
| **EDM festival** | Êxtase coletivo, drop coreografado, mass experience |
| **Lo-fi study** | Background ambient, produtividade calma |
| **Bossa Nova BR** | Sofisticação suave, cool melancólico |
| **Funk carioca BR** | Energia visceral, vocabulário comunitário |
| **Sertanejo BR** | Histórias de povo, narrativa universal |
| **MPB clássica** | Sofisticação intelectual + popular |

## Como skill usa

- Quando briefing pede **emoção/intensidade/identidade tribal** → puxa daqui
- Quando briefing está num domínio "frio" (tech, B2B, financeiro) → puxa daqui pra esquentar
- Quando precisa de **estrutura narrativa em arco** → ABA, build-drop, bridge
- Para **naming** → Blend linguístico com termos musicais (Tempo, Cadenza, Acorde, Coda, Sustain)
