# Léxico cross-domain — RELIGIÃO / RITUAL

> Vocabulário e mecânicas religiosas/rituais como matéria-prima pra Bend/Blend. Domínio poderoso porque opera em **devoção** e **identidade tribal** — territórios raros em marketing.

> ⚠️ **Sensibilidade:** uso de iconografia religiosa direto pode ofender. Skill prefere **mecânicas estruturais** (rito, comunidade, hierarquia) sobre **símbolos** (cruz, lua crescente, mantra).

## Dimensões centrais

- **Devoção** (frequência, intensidade, ritual)
- **Hierarquia** (sacerdote, fiel, neófito, sumo)
- **Sacralidade** (sagrado vs profano, tabu)
- **Comunidade** (congregação, tribo, irmandade)
- **Texto** (escritura, dogma, parábola)
- **Tempo** (calendário sagrado, jejum, festa)
- **Espaço** (templo, altar, peregrinação)
- **Transformação** (conversão, iluminação, redenção)

## Como skill usa

- Quando briefing precisa **devoção / lealdade extrema**
- Quando precisa **estrutura de comunidade hierarquizada**
- Quando precisa **ritual recorrente que cria identidade**
- Para **marca de transformação pessoal** (saúde mental, fitness, educação)
- Para **estética sagrada** (com cuidado cultural)
- Para **naming**: Sanctum, Cenacle, Litany, Vow, Apostle, Cathedral, Prism (luz colorida vidraça)

## Avisos

- Religião é território **emocional**. Bend/Blend bem feito = poderoso. Mal feito = ofensivo.
- Prefira **mecânicas estruturais** (rito, comunidade, hierarquia) sobre **símbolos específicos** (cruzes, mantras).
- Quando símbolo é apropriado, exige curadoria humana (CCO + CLO consultados antes de implementar).
