# Zeus-CO — Cérebro Criativo

**API criativa transversal do Zeus-CO.** Skill operadora que GERA ideias aplicando neurociência da criatividade unificada com 4 frameworks acadêmicos formais.

## Princípio fundador

> "Criatividade não é magia — é uma operação cognitiva específica do córtex pré-frontal, com 3 vetores formalizáveis (dobrar/quebrar/misturar), 2 amplificadores (software da inovação + memória-atenção) e métrica matemática de originalidade (distância semântica)."

## Por que existe

LEPs Chiefs (CEO/CFO/CMO/CCO/CTO/COO/CLO) e Specialists frequentemente precisam destravar pensamento divergente — não pra delegar a alguém "criativo", mas pra **executar** um movimento criativo dentro da própria função. Esta skill é o motor compartilhado.

## Fundamentação acadêmica (não é vibes)

Síntese unificada de:
- **Eagleman** (*The Runaway Species*) — Bend, Break, Blend operacional
- **Boden** (3 types) — Combinatorial / Exploratory / Transformational
- **Fauconnier-Turner** (Conceptual Blending Theory) — formalização matemática de Blend
- **Hofstadter** (Copycat / FARG slipnet) — arquitetura cognitiva pra analogia (Bend)
- **Gärdenfors** (Conceptual Spaces) — geometria de conceitos
- **Beaty & Johnson** (SemDis 2021) — distância semântica como métrica de originalidade
- **Eberle/Osborn** (SCAMPER) — operacionalização concreta dos 3 vetores
